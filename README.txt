README
In order for this program to run, here are the steps that must be taken:
1. Open ZIP file
2. Run extract_words.py to create the text files (allword.txt, uniquewords.txt, and wordfrequency.txt), based on the Dracula text
3. In order to view the word cloud, click on wordcloud.PNG 
4. To run a3_novelvisualization, the a3_novelvisualization folder contains the a3_novelvisualization.pde file which will run,
    this file also contains a Trattatello.ttf font file that can be installed if this is not already available on the system. 
5. To run a3_wordfrequency, open the a3_wordfrequency.pde file which should be under the a3_wordfrequency directory and press run.
